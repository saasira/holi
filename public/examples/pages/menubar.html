Menubar Variants

Variant 1: component="menubar"

Desktop: top-level menu is horizontal with hover/focus submenus. At and below the breakpoint, it collapses to a labeled button (set with data-mobile-toggle="button").

Variant 1b: bar mode open-on-click

Desktop touch-friendly variant: open submenus on click (not hover). Also auto-collapses to the same mobile menu button at the configured breakpoint.

Variant 2: role="menubar"

Role-based variant using icon collapse mode with data-mobile-toggle="icon" for the classic hamburger trigger.

Variant 3: <menubar single-button>

Single menu button mode opens/closes the root menu on click, with nested items still collapsed by default.
